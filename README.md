# BattleShips

## Description
This is a simple battleships game made with Next.js and React. It is a work in progress.

## How to Use

### Install Dependencies

```bash
npm install
```

### Run Dev Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Run Production Server

```bash
npm start
```

## Run Jest Tests

```bash
npm test
```
